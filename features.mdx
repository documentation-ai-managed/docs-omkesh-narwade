## Skills Section

Display your skills prominently to grab recruiters' attention. The template supports progress indicators, icons, and categorization.

<Callout kind="tip">
  Prioritize top skills like `JavaScript`, `React`, and `AWS` at the top for maximum impact.
</Callout>

Customize the skills array in your config file:

<CodeGroup tabs="JavaScript,TypeScript">
  ```javascript
  const skills = [
    { name: 'JavaScript', level: 90, category: 'Frontend' },
    { name: 'Python', level: 85, category: 'Backend' },
    { name: 'React', level: 95, category: 'Frameworks' },
    { name: 'AWS', level: 80, category: 'Cloud' }
  ];
  ```
  ```typescript
  interface Skill {
    name: string;
    level: number;
    category: string;
  }

  const skills: Skill[] = [
    { name: 'JavaScript', level: 90, category: 'Frontend' },
    { name: 'Python', level: 85, category: 'Backend' },
    { name: 'React', level: 95, category: 'Frameworks' },
    { name: 'AWS', level: 80, category: 'Cloud' }
  ];
  ```
</CodeGroup>

## Education Background

Highlight your MIT Aurangabad degree and certifications. Include timelines, GPAs, and achievements.

<Steps>
  <Step title="Add Your Degree" icon="graduation-cap">
    Edit the `education` section in `data/profile.ts`:

````typescript

  {
    institution: 'MIT Aurangabad',
    degree: 'B.Tech Electronics and Computer Engineering',

    gpa: '9.2/10'
  }
````

  </Step>


  </Step>
