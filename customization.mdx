## Update Personal Information

Edit the `data/profile.json` file or relevant MDX components to update your bio, education, and skills. Locate the file in your project root.

<Steps>
  <Step title="Locate Config" icon="search">
    Open `data/profile.json`.
  </Step>
  <Step title="Edit Details" icon="edit">
    Update fields like `name`, `bio`, and `education`.

````json
{
  "name": "Omkesh Narwade",
  "bio": "Electronics and Computer Engineering student at MIT Aurangabad with expertise in programming, web technologies, and cloud computing.",
  "education": [
    {
      "degree": "B.Tech Electronics and Computer Engineering",
      "institution": "MIT Aurangabad",
      "year": "2025"
    }
  ]
}
````
  </Step>
  <Step title="Save and Preview" icon="eye">
    Save changes and run `npm run dev` to preview.
  </Step>
</Steps>

<Callout kind="tip">
  Use markdown in `bio` for formatting, like **bold** skills or *italics* for emphasis.
</Callout>

## Customize Color Scheme

The default brand color is `#3B82F6`. Override CSS custom properties in `src/styles/globals.css` or Tailwind config for consistent theming.

<Tabs>
  <Tab title="CSS Variables" icon="css3">
    Edit root variables.

````css
:root {
  --brand-color: #3B82F6;
  --brand-hover: #2563EB;
}

.button {
  background: var(--brand-color);
}
````
  </Tab>
  <Tab title="Tailwind Config" icon="tailwind">
    Extend theme in `tailwind.config.js`.

````javascript
module.exports = {
  theme: {
    extend: {
      colors: {
        brand: {
          500: '#3B82F6',
          600: '#2563EB'
        }
      }
    }
  }
};
````
  </Tab>
</Tabs>

<Callout kind="alert">
  Test color contrast with tools like WebAIM to ensure accessibility. Avoid low-contrast combinations.
</Callout>

## Integrate External Links

Connect your GitHub and LinkedIn profiles in the header or footer components. Edit `src/components/Header.jsx` or `data/social.json`.

<Expandable title="Social Links Config" default-open="true">
  Update links for seamless navigation.

````json
{
  "social": [
    {
      "platform": "GitHub",
      "url": "https://github.com/omkeshnarwade",
      "icon": "github"
    },
    {
      "platform": "LinkedIn",
      "url": "https://linkedin.com/in/omkesh-narwade",
      "icon": "linkedin"
    }
  ]
}
````

  Use `target="_blank"` for external navigation.
</Expandable>
