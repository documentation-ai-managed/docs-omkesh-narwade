## Recent Updates

Stay informed about the latest changes to the Omkesh Narwade portfolio starter kit. You receive new features, performance improvements, and bug fixes with each release. Review the details below to understand what's new and how it impacts your site.

## Upgrade Guide

Follow these steps to update your portfolio site to the latest version. You back up your customizations first to avoid losing changes.

<Steps>
  <Step title="Backup Your Site" icon="save">

  Create a backup of your current repository.

  ```bash
  git add .
  git commit -m "Backup before upgrade"
  git push origin main
  ```

  </Step>
  <Step title="Update Dependencies" icon="package">

  <Tabs>
    <Tab title="npm" icon="npm">
    ```bash
    npm install @omkeshnarwade/portfolio-starter@latest
    ```
    </Tab>
    <Tab title="yarn" icon="yarn">
    ```bash
    yarn add @omkeshnarwade/portfolio-starter@latest
    ```
    </Tab>
    <Tab title="pnpm" icon="package">
    ```bash
    pnpm add @omkeshnarwade/portfolio-starter@latest
    ```
    </Tab>
  </Tabs>

  </Step>
  <Step title="Run Migrations" icon="settings">

  Check for any database or config migrations.

  ```bash
  npm run migrate
  ```

  Review `CHANGELOG.md` for breaking changes.

  </Step>
  <Step title="Deploy" icon="rocket">

  Build and deploy your updated site.

  ```bash
  npm run build
  npm run start
  ```

  </Step>
</Steps>

<Callout kind="tip">

Subscribe to the [GitHub releases page](https://github.com/omkeshnarwade/portfolio-starter/releases) for notifications on new versions. You enable them in your GitHub settings.

</Callout>
